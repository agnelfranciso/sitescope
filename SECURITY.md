# Security Policy 🛡️

SiteScope is an open-source tool, and we take security and privacy seriously. 

## 🔭 Vulnerability Reporting
Please **do not** open an issue on GitHub for security vulnerabilities. Instead, report them privately (if you'd like to reach the developers, feel free to contact us through our personal social channels or website).

## 🔭 Privacy
- SiteScope is a **local terminal tool**. It does not send any of your website data or screenshots to any external servers. 
- All screenshots and reports are stored locally in your current output directory.

## 🔭 Supported Versions
We recommend always using the latest version of SiteScope to ensure you have the latest security patches and browser engine (Playwright) updates.
To check for updates, simply run the **`INSTALL_SITESCOPE.bat`** in your project folder.
